# interlacing_numbers
The notebook "calculate_ilns.ipynb" contains code that choses randomly selected trig polynomials, calculates the periodic maximizing measure out of all periodic orbits up to period 14 (represented by Lyndon words), then calculates interlacing number of each maximizng measure. It also contains a simple visulatization of interlacing number for a given Lyndon word by plotting the orbit and antiorbit on the unit circle.
The notebook uses the list stored in "Lyndon Words.txt" as the list of candidate measures.
